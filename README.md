# 💊 Epos Pharma

> A modern Electronic Point of Sale (EPOS) system built for pharmacy management and retail operations.

## 📖 Overview

**Epos Pharma** is a purpose-built Electronic Point of Sale (EPOS) system tailored for pharmacies and medical retail stores. It streamlines day-to-day operations — from product inventory and sales processing to billing and stock management — all within a clean, easy-to-use interface.

Designed with pharmacists and medical store owners in mind, Epos Pharma reduces manual errors, speeds up the checkout process, and keeps your inventory under control.

---

## ✨ Features

- 🛒 **Point of Sale Interface** — Fast and intuitive checkout process for pharmacy staff
- 💊 **Medicine Inventory Management** — Add, update, and track medicine stock with ease
- 🧾 **Billing & Receipts** — Generate itemized bills and digital receipts for customers
- 📦 **Stock Tracking** — Real-time visibility into inventory levels and low-stock alerts
- 🔍 **Product Search** — Quickly search medicines by name or category
- 📊 **Sales Records** — View and manage transaction history
- 🖨️ **Receipt Printing** — Print-ready billing output for physical receipts

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v16 or higher)
- npm (comes with Node.js)

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/RehanAslam2004/Epos-Pharma.git

# 2. Navigate into the project directory
cd Epos-Pharma

# 3. Install dependencies
npm install

# 4. Start the application
node server.js
```

### Access the App

Open your browser and go to:

```
http://localhost:3000
```
